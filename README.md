# 📦 swap

> Temporary file sharing for the Franq team.

This repository is used exclusively for **short-term file exchange** between coworkers. It is not a long-term storage solution — files here are transient and may be removed at any time without prior notice.

---

## 🚀 How to use

1. **Upload** your file by adding it to the repository (via the GitHub UI or a `git push`).
2. **Share** the direct file URL with your coworker.
3. **Clean up** — delete the file once it has been downloaded or is no longer needed.

---

## 📋 Guidelines

- ✅ Use for temporary sharing of documents, exports, assets, and other work files.
- ✅ Keep file names descriptive (e.g., `relatorio-abril-2026.pdf`, not `arquivo1.pdf`).
- ❌ Do **not** store sensitive or confidential data (passwords, API keys, PII, client data).
- ❌ Do **not** use this as a permanent archive — use the appropriate internal systems for that.
- ❌ Do **not** commit large binary files (>50 MB). Use a cloud storage link instead.

---

## 🗂️ Structure

There is no enforced folder structure. Feel free to organize by date or project if it helps, but keep it simple:

```
swap/
├── your-file.pdf
├── export-2026-03.csv
└── ...
```

---

## ⚠️ Disclaimer

Files in this repository are **not backed up** and can be deleted at any time. Do not rely on this repo for permanent storage. Always keep your own copy of any important files you share here.
